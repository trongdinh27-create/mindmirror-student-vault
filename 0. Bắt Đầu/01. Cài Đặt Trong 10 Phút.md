# Cài đặt trong 10 phút

## 1. Mở vault

Trong Obsidian chọn **Open folder as vault** và chọn thư mục `MindMirror Student Vault Template`.

## 2. Bật plugin

Toàn bộ plugin đã được đóng gói trong vault. Vào **Settings → Community Plugins**, tắt Restricted Mode và reload Obsidian. Năm plugin cốt lõi cần được bật là:

- Templater
- Dataview
- Calendar
- Periodic Notes
- Excalidraw

Các plugin bổ trợ cũng đã được đóng gói và khai báo sẵn. Sau khi mở vault, nên cập nhật từ Community Plugins để nhận phiên bản mới và điều khoản bảo mật của tác giả.

## 3. Chạy setup

```bash
python3 scripts/setup.py
```

Script chỉ tạo `Me.md` khi chưa có, không ghi đè dữ liệu.

## 4. Kiểm tra nhanh

- Tạo note mới: note phải vào `1. Thu Thập/`.
- Mở Daily Note: note phải vào `2. Tinh Lọc/Nhật Ký Ngày/`.
- Kéo ảnh vào note: ảnh phải vào `5. Hộp Công Cụ/Tệp Đính Kèm/`.
- Tạo note trong các folder cốt lõi: Templater phải chèn đúng Template.

## 5. Điền Me.md

`Me.md` là lớp ngữ cảnh giúp AI hiểu bạn. Không chia sẻ file này nếu có dữ liệu riêng tư.
