# 3. Chuyển Hoá

Nơi tri thức đủ chín được kết tinh để dùng lại lâu dài.

- `Bản Đồ/`: địa chỉ và điều hướng.
- `Kiến Thức Cốt Lõi/`: câu khẳng định hoàn chỉnh, có nguồn và liên kết.
- `Framework/`: quy trình hoặc mô hình có thể lặp lại.
