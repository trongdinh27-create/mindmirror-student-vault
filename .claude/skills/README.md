# MindMirror Skills

Đây là nguồn workflow canonical, dùng được bởi Claude Code và các client hỗ trợ Agent Skills.

- Mỗi skill nằm trong một thư mục và có `SKILL.md`.
- Dùng đường dẫn tương đối; không ghi dữ liệu cá nhân hoặc đường dẫn máy.
- `.agents/skills/` chỉ chứa adapter Codex.
- Hermes wrapper được tạo bằng `python3 scripts/install_hermes_wrappers.py`.
